# Promods-DL
Télécharger facilement et automatiquement les fichiers Promods

T'en as marre d'attendre 5 min entre les téléchargement?
T'en as ras le cul de vouloir jouer, mais que tu doives attendre 3h pour jouer en multi? 

> Cette extension est faite pour toi!

Il suffit que tu te rendes sur le site officiel de promods, tu fais l'install de ton fichier SBS, tu choisis l'installation standard, puis tu lances le dl, via la carte ajoutée sur le site grâce à l'extension.

Bonne journée et bon jeu!
